# Anna Parker — Portfolio

**Live site:** https://anp-exe.github.io/anna/

## Structure

```
index.html                 The entire site (HTML + inline CSS)
images/                    Assets — hero portrait, lace frame, badge logos, banner, favicon
Anna-Parker-CV.pdf         One-page CV (AI/ML internship)
Anna-Parker-CV.docx        Editable CV
.github/workflows/pages.yml  Auto-deploy to GitHub Pages
```

## Featured projects

- **Women in Chess** — an interactive data story analysing 545k FIDE players with a Prophet
  time-series pipeline and Monte Carlo simulation. *Codédex Staff Pick.*
- **AI Slop Detector** — a Python tool scoring how AI-generated a post reads via a hybrid
  rules + zero-shot Hugging Face ensemble. *Codédex Monthly Challenge winner.*
- **Care Leavers** — a data-journalism scrollytelling piece built from DfE and UCAS datasets.

## Deployment

The site is static and deploys automatically to GitHub Pages via GitHub Actions on every
push to `main` (see `.github/workflows/pages.yml`). To run locally, just open `index.html`
in a browser.
